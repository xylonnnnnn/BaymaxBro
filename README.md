# Baymax Health Bot

Telegram-бот для анализа базовых показателей здоровья. Бот помогает считать ИМТ, вести дневной трекер воды, анализировать сон, строить план КБЖУ и смотреть статистику.

## Возможности

- Расчет ИМТ с сохранением истории.
- Трекер воды с дневным учетом:
  - можно добавить воду к текущему значению за день;
  - можно заменить количество воды за день.
- Анализ сна:
  - продолжительность сна;
  - оценка сна;
  - история и график.
- План КБЖУ:
  - расчет калорий, белков, жиров и углеводов;
  - сохранение пола, возраста, роста, веса и уровня активности;
  - цель выбирается каждый раз отдельно.
- Общая статистика за день:
  - профиль пользователя;
  - текущий ИМТ;
  - вода за день;
  - сон за день;
  - актуальный план КБЖУ;
  - список недостающих данных и функций, которые стоит пройти.
- SQLite-база данных для пользователей, профиля и истории.
- Графики статистики через `matplotlib`.

## Структура проекта

```text
.
├── app/
│   ├── handlers.py      # обработчики Telegram-бота
│   └── keyboards.py     # inline-клавиатуры

├── database.py          # SQLite-схема и запросы
├── stats_graphs.py      # генерация PNG-графиков
├── config.py            # токен бота
├── run.py               # запуск бота
└── requirements.txt
```

## База данных

При запуске бот автоматически создает файл:

```text
baymax.sqlite3
```

Основные таблицы:

- `users` - Telegram ID пользователей.
- `user_profiles` - сохраненные данные профиля: пол, возраст, рост, вес, активность, средний сон.
- `bmi_entries` - история ИМТ.
- `water_entries` - дневная история воды.
- `sleep_entries` - история анализа сна.
- `kbzhu_entries` - история планов КБЖУ.

## Установка

Создать виртуальное окружение:

```bash
python3 -m venv .venv
```

Активировать его:

```bash
source .venv/bin/activate
```

Установить зависимости:

```bash
pip install -r requirements.txt
```

## Запуск

```bash
.venv/bin/python run.py
```

После запуска бот инициализирует SQLite-базу и начнет polling.

## Проверка

Проверить синтаксис проекта:

```bash
.venv/bin/python -m compileall run.py app functions advices database.py stats_graphs.py
```

Проверить импорт обработчиков:

```bash
.venv/bin/python -c 'import app.handlers; print("handlers import ok")'
```

## Заметки

- Вопросы бота удаляются после ответа пользователя, чтобы в чате оставались отчеты, меню и графики.
- Для воды одна запись соответствует одному дню.
- Для общей статистики вода и сон считаются именно за текущий день.
- ИМТ и КБЖУ в общей статистике используются из актуальных сохраненных данных, если они уже есть.
